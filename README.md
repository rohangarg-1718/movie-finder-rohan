# Movie Finder

A responsive Movie Discovery App built with Next.js and OMDb API.

## Features

* Search movies in real time
* Responsive movie grid
* Movie details page
* Favorites with localStorage persistence
* Loading, error, and empty states
* Manual pagination with Next/Previous controls
* Responsive design for mobile and desktop

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* OMDb API

## API Used

This project uses the OMDb API as the movie data source. OMDb was chosen as an alternative public movie API and is used for movie search, details, and metadata retrieval.


## Run Locally

```bash
npm install
npm run dev
```

Open:

http://localhost:3000

## Deployment

Deployed using Vercel.
